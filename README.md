# Digital Clock

A simple digital clock built using HTML, CSS, and JavaScript.  
It displays the current time in hours, minutes, and seconds and updates every second.  
The clock also shows the current date and supports switching between 12-hour and 24-hour formats.

## Features
- Live updating time (HH:MM:SS)
- 12-hour and 24-hour format toggle
- Shows AM/PM indicator (for 12-hour mode)
- Displays full date
- Option to show or hide seconds

## Technologies Used
- HTML5
- CSS3
- JavaScript (ES6)

## How to Run
1. Download or clone the repository.
2. Open `index.html` in any modern web browser.
3. The clock will start automatically.

## Project Structure
- `index.html` – UI structure
- `style.css` – styling for the clock interface
- `script.js` – logic for updating time and controlling formats
